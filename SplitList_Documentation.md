# SplitList iOS Grocery App

## Introduction
SplitList is an iOS application designed to help users manage their grocery shopping efficiently. The app allows users to create lists, categorize items, and share lists with others for collaborative shopping experiences.

## Setup Instructions
1. **Clone the Repository**: Use the command `git clone https://github.com/kaanvict/Claude.git` to clone the repository to your local machine.
2. **Install Dependencies**: Navigate to the project directory and run `pod install` to install the required dependencies.
3. **Open Xcode**: Open the project using Xcode by opening the `.xcworkspace` file.

## Usage Guidelines
- **Creating a List**: Users can create a new grocery list by tapping the plus icon on the home screen. 
- **Adding Items**: To add items, select the list and use the add item button.
- **Collaborative Shopping**: Users can share their lists via email or messaging apps.
- **Categorization**: Items can be categorized (e.g., fruits, vegetables, dairy) for better organization.

## Nomenclature
- **List**: A grouping of grocery items that users can manage in the app.
- **Item**: Individual grocery products that can be added to a list.
- **Category**: A classification used to organize items in a list.
